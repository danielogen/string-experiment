Practice Tasks

Before you start the actual experiment, we thought it is a good idea to practice with a few tasks. There are four tasks and you can repeat as many times as you need. The symbol "+" is for concatenation and "{ }" for interpolation (embedding). Read the task and select THE OPTION YOU THINK IS CORRECT by pressing 1, 2, 3 or 4. Once you feel confident to start the actual experiment, click "PROCEED" at the button right corner of practice task 4 or press "ENTER / RETURN" key on your keyboard. To repeat the tasks, simply press RIGHT ARROW KEY.

Task 1 of 4: The "+" symbol represents concatenation. Press 1, 2, 3 or 4 to select the correct concatenation. 

              
                a = "Hello "
                b = "World"

                a + b
Options:
  1. "HelloWorld"
  2. "Hello World"
  3. "World "
  4. "There"

Task 2 of 4: The "{ }" symbol represents interpolation. Press 1, 2, 3 or 4 to select the correct interpolation.

          
            a = "Hello "
            b = "World"

            "{b}{a}"
Options:
  1. "WorldHello "
  2. "Hello World"
  3. "World "
  4. "There"

Task 3 of 4: The "+" symbol represents concatenation. Press 1, 2, 3 or 4 to select the correct concatenation.

          
              a = "Hi"
              b = "There"
              c = 4
              
              c + b + a
Options:
  1. "Hi There 4"
  2. "Hi 4 There"
  3. "4 There Hi "
  4. "4 Hi There"

Task 4 of 4: The "{ }" symbol represents interpolation. Press 1, 2, 3 or 4 to select the correct interpolation.

      
          a = "Hi"
          b = "There"
          c = 4
          
          "{b} {a} {c}"
Options:
  1. "Hi There 4"
  2. "There 4 Hi"
  3. "4 There Hi "
  4. "There Hi 4"
